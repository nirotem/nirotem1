# Personal Academic Website - Nir Rotem

This is a clean, modern academic website built with pure HTML and CSS.

## Folder Structure
- `index.html`: About/Home page.
- `projects.html`: Research projects.
- `publications.html`: Zotero integration via BibBase.
- `teaching.html`: Teaching history and conferences.
- `lighter.html`: Personal interests.
- `style.css`: The central design file.
- `images/`: Folder for all images.
  - `profile.jpg`: Your profile picture.
  - `graph-name.png`: Placeholders for project graphs.

## How to Edit
1. **Text**: Open any `.html` file in a text editor and change the text inside the tags.
2. **Images**: Replace files in the `images/` folder with your own files. Make sure the filenames match the code.

## How to Deploy to GitHub Pages
1. Create a new repository on GitHub named `yourusername.github.io`.
2. Upload all files to this repository.
3. Your site will be live at `https://yourusername.github.io`.
